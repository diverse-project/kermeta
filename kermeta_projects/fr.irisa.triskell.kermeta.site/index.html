Update and discovery URL to be used in Eclipse : http://www.kermeta.org/update
Using this system you can easily install and keep your version of KerMeta workbench up-to-date.
It includes:
  • KerMeta interpreter
  • KerMeta perspective, editor and outliner
  • KerMeta conversion to/from ecore

To install these plugins, point your Eclipse Update Manager at this site. For more on how to do this, click here. 

   Help
    > Software Updates
       > Find and Install...
          > Search for new features to install
             > Add Update Site...
                * Name: KerMeta Update Manager Site for Eclipse
                * URL:    http://www.kermeta.org/update 




Note: If you don't see the links to the jar files, this is due to browser incompatibility. In this case, please try with another browser. Sorry for the inconvenience.
