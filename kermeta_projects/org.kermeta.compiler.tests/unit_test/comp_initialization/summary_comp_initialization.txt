Design:

- Initialization of a variable:

In kmt: var aPrinter : Printer init Printer.new
In Java: comptest2.Printer aPrinter = comptest2.Comptest2Factory.eINSTANCE.createPrinter();

- Initialization of a String:

In kmt: var text1 : String init "Hello"
In Java: String text1 = "Hello ";

In kmt:
	var text2 : String
	text2 := "World"
In Java:
	String text2 = null;
	text2 = "World";
