#readme.txt, org.kermeta.compiler.kmt

This project contains the source code in Kermeta of the compiler.

-The main class and operations are in org.kermeta.compiler.kmt/src/kermeta/StructureCompiler.kmt.
-The resulting files are: *.ecore, *.simk, ready for the Java generation.
-Before launch the execution, that is required to generate the km merged and typechecked of your kmt program.

The folders models, fsmmodels and test are resources that are used to develop the compiler.
Many resources (*.km, *.traceability, *.simk) into these folders may be deprecated.
