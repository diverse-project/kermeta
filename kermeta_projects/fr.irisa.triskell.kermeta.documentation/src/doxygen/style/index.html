Kermeta project documentation

Welcome to the Doxygen documentation of Kermeta projects. You will find here :
  • Framework project: it contains the code used to compile Kermeta framework.
  • Interpreter project: it contains the java program that interpretes Kermeta (parsing, runtime memory loading, interpretation)
  • IO project: it contains facilities that convert Kermeta code to another format (mainly ecore, compiled kermeta -- xmi) and vice versa
  • Runner project: the Eclipse plugin to run or debug a Kermeta program
  • Texteditor project: the Eclipse plugin to edit a Kermeta program
