Preliminary note: the html directory contains the files that are necessary to 
generate kermeta API documentation in HTML.

There is only one HTML file that is generated by a Java program that you will find 
here : 
fr.irisa.triskell.kermeta.documentation/
	src/java/fr.irisa.triskell.kermeta.documentation/KM2HTMLMain.java
	
To generate the Kermeta API/Framework documentation, you just have to run this KM2HTMLMain.java.
To change the default configurations (input and output files), open it. It is easy to change it
manually in the code :)
	
The generated HTML file will be stored in the "src/html" directory, and be named:
"framework.km.html".

The program uses the following files :

 * in fr.irisa.triskell.kermeta project (accessed through a relative path, so, be careful!):
        lib/framework.km : the kermeta framework/api
 * in this project src/html : 
 		screen.css : the main stylesheet
 		header.html : the header (<head></head>)
 		framework.km.html : this is the generated file
