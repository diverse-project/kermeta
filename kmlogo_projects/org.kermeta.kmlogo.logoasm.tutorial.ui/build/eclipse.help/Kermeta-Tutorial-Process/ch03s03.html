3.3. Textual editor

Textual editor allows you to "program" your metamodel. Instead of designing it with its representation, you 'll textually define the concepts of your metamodel. Then, with Kermeta, you obtain the ecore file by calling the generate ecore metamodel on your Kermeta program (.kmt).

Note

If you want to have an example of how to "program" the abstract syntax please refer to the example in
