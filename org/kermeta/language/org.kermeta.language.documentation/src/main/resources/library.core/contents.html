library.core

This is the main dependency of every Kermeta program. It provides the basic types and the core framework for manipulating models. It also provides Kermeta metamodel itself.

In order to use it, you must have the following dependency in your project kp file: 

dependency "library.core" = "platform:/plugin/org.kermeta.language.library.core", 
                            "mvn:org.kermeta.language/language.library.core/2.0.1-SNAPSHOT"
